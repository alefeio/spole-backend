# Módulos por domínio

Esta pasta contém os módulos de domínio da API (monólito modular).

Na Sprint 00, as pastas são apenas **marcadores de estrutura** para evolução nas próximas sprints, sem rotas nem lógica de negócio.

Cada domínio deverá, quando aplicável, seguir a organização prevista em `docs/00-product/api-standards.md`:

- `routes`
- `controller`
- `service` (ou `use-cases`)
- `repository`
- `schema`/`validator`
- `dto`
